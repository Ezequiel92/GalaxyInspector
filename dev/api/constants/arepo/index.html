Arepo constants

This is a list of constant and types used in GalaxyInspector, specific for the Arepo codebase.

None of these constants and types are exported.

GalaxyInspector.ELEMENT_INDEX — Constant

Code index for each tracked element.

source
GalaxyInspector.GC_BASENAME — Constant

Base name of the group catalog files.

source
GalaxyInspector.HYDROGEN_MASSFRAC — Constant

Mass fraction of hydrogen.

source
GalaxyInspector.METAL_LIST — Constant

List of element indices above helium.

source
GalaxyInspector.PARTICLE_INDEX — Constant

Current cell/particle index in use.

source
GalaxyInspector.PARTICLE_NAMES — Constant

Current human readable name of each cell/particle type in use.

source
GalaxyInspector.QUANTITIES — Constant

Dictionary of dimensional properties for the quantities in the code.

source
GalaxyInspector.SNAP_BASENAME — Constant

Base name of the snapshot files, set in the code variable SnapshotFileBase.

source
GalaxyInspector.SOLAR_METALLICITY — Constant

Solar metallicity, as used in Arepo.

References

M. Asplund et al. (2006). The new solar abundances - Part I: the observations. Communications in Asteroseismology, 147. doi:10.1553/cia147s76

source
GalaxyInspector.TRACER_MASS — Constant

Mass of the tracers in internal code units. Its value comes from All.TargetGasMass = All.TargetGasMassFactor * All.ReferenceGasPartMass in the code.

source
