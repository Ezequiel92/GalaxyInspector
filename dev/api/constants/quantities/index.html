GalaxyInspector.parseHaloQuantity — Method
parseHaloQuantity(s::Symbol)::Tuple{Symbol,Int}

Takes a symbol for an halo (FoF group) quantity in the format :quantity_haloidx and splits it into the text part as a Symbol and the number part as an Int.

Arguments

  • s::Symbol: Target halo and halo quantity, e.g., :halo_mass_12. The index of the target halo starts at 1 and the halo quantity has to be one of the keys of HALO_KEYS.

Returns

  • A Tuple with two elements:

    • A Symbol for the target quantity.
    • The index of the target halo.

Examples

julia> parseHaloQuantity(:halo_mass_12)
(:halo_mass, 12)

julia> parseHaloQuantity(:halo_M200_1)
(:halo_M200, 1)
