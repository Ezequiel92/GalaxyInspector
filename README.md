A Julia module for data analysis of Gadget2/3/4 and Arepo simulations.

There are more advanced tools to analyze/plot the data generated by Gadget (you can see some in the [References](https://github.com/Ezequiel92/GadgetInspector#-references) section), but this module was written, not only as a basic plotting tool, but as an exercise to learn [Julia](https://julialang.org/) and software development in general (testing, documentation, version control, etc.).

## ⚙️ Characteristics

- This works with the snapshots in binary format (option `SnapFormat = 2`) or in [HDF5](https://www.hdfgroup.org/solutions/hdf5/) (option `SnapFormat = 3`).
- This is a script inside a module, not a package. Global constants and data structures are defined in `src/constants.jl`.
- The dependencies are given by the `Project.toml` file.

## 📘 Examples

Examples can be found in the repository [plotting_scripts (🚧 WIP 🚧)](https://github.com/Ezequiel92/plotting_scripts).
